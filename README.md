# Cosmic Byte CB-GK-37 Reverse Engineered Led Protocol

Reverse engineering the LED protocol for the Cosmic Byte CB-GK-37 keyboard. This repository contains code written primarily in Rust and Python for interfacing with and controlling the keyboard's LED functionalities.

## Language Composition

- **Rust (82.5%)**: The majority of the implementation is done in Rust, ensuring performance and safety in low-level interfacing with the hardware.
- **Python (17.5%)**: Python is used for scripting, testing, or high-level communication with the hardware.

## Repository Overview

This repository is dedicated to reverse-engineering the LED protocol of the Cosmic Byte CB-GK-37 keyboard. The work done here can enable the development of custom software to control the keyboard's lighting effects and patterns.

### Features

- Reverse-engineered LED protocol for the CB-GK-37 keyboard.
- Code to interact with the keyboard's lighting system.
- Modular design for easy extension and integration.

## Getting Started

### Prerequisites

To work with this repository, ensure you have the following installed:

- **Rust**: For compiling and running the Rust modules.
- **Python**: For running the Python scripts.

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/sipusumit/cbkbd.git
   ```
2. Navigate to the project directory:
   ```bash
   cd cbkbd
   ```
3. Build the Rust components:
   ```bash
   cargo build --release
   ```
